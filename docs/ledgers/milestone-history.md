## Historical Scope Notes

- Phase 1 completion means the current named target set is green against the relevant oracle; it does not mean bit-for-bit librime feature parity.
- TypeDuck behavior remains profile-specific. Default core behavior and default `RimeApi` follow upstream `rime/librime 1.17.0`.
- Browser-visible claims require real browser evidence. Native engine wins are not browser startup or typing wins unless a rebuilt browser path proves them.
- Finished plans move to [`plans/completed/`](../plans/completed) and remain the detailed historical record.
